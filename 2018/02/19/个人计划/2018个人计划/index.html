2018个人计划

发表于 2018-02-19 | 分类于 个人计划

原文网址QQ空间日志

一、工作上的计划：

  • 1.0争取4月份能够完成口碑课程1.0的测试版本，能够在苹果商店上线。
  • 1.1我必须要学会口碑课程项目的所有功能实现的技术难点。
  • 1.2 我要学会如何在苹果商店发布上架，打包app一系列技术。
  • 1.3 关于swift也要去学，相比较object-c 有它的优点。
  • 1.4 争取三月份能够转正成功，继续在ios岗位奋斗学习。
  • 1.5 7月拿到毕业证后，我希望那时我的技术值5000块，能够有资格和公司老板说，我要涨工资，不涨工资就下一家。
  • 1.6 多与同事交流，锻炼自己的沟通能力，表达能力，内向的性格，学会如何和公司的领导说话，处理同事关系。
    1.7 不要迟到，不要早退，上班时间里，拿出效率来，适应工作节奏，并且驾驭自己的工作节奏。
    1.8 做事要和领导沟通，要让他们知道，你在很努力的做他分配给你的事。

二、个人学习计划：

  • 1.0听、读微信读书app里的《你要么出众要么出局》、《你所谓的稳定不过是在浪费生命》、《我不过低配的人生》、《从行动开 始》、《iOS移动开发从入门到精髓》、《逻辑思维》、《人际沟通关键法则》
  • 1.1 上班坐公交车的接近一个小时时间用来看这些书，下班回家的时间，回顾今天工作怎么样，什么地方需要改进，什么地方需要继续保持，多的时间也用来看书，看看时事咨讯。
  • 1.2 回家每天写工作日记，写明天的工作计划安排，温习今天工作中学到的新东西，再次剖析知识，变成自己的东西，融进我的知识体系里。
  • 1.3 关注免费的ios视频直播，觉得自己需要的，能对自己有提升的，晚上就看看直播。
  • 1.4 抽时间，了解python这门语言，人工智能相关的知识，自己要往人工智能上面靠。
  • 1.5 学会10首自己爱听的歌，能够做到开着伴奏唱出来有感觉。
  • 1.6 学习swift 这门编程语言，能够写一个简单的应用程序。
  • 1.7 经营管理好自己的博客，经营几个GitHub上的项目，看大神们的博客，学习借鉴他们如何成长的。
  • 1.8 认真准备4月份的专衔本考试，争取一次通过，能够达到写论文的要求。

三、个人生活计划：

  • 1.0 学会穿衣搭配，让自己变帅，不要再邋里邋遢的去工作，对自己职场也有帮助。
  • 1.1 发型可以的话，怎样帅就弄成怎样，从小没有尝试过烫发，想体验一次。
  • 1.2 休息的时候两个星期一次，带着女朋友出去玩，因为她不想走远了，打算重庆的景点挨个逛。
  • 1.3 多和父母交流自己的想法，看法，认知，让他们也进步，多了解我。
  • 1.4 和亲人多联系，爷爷奶奶打算一个月至少一次通电话，和哥哥姐姐们多在群里互动一下，和公司的同事多交流一下，分享自己的所得，分享自己的经验教训。
  • 1.5 尝试自己带饭去公司吃。
  • 1.6 我想4月份考试过啦，我考虑去把驾照学啦，虽然工作了，时间少，但是还是觉得应该去学个驾照，自家有个车还是方便很多。
  • 1.7 结交几个志同道合的朋友，一起学习进步。
  • 1.8 能参加同学聚会，公司聚会，私人聚会，就去，这是很好的了解彼此，锻炼能力的机会。
  • 1.9 多为家里做做事，为爸妈分担，学怎么炒菜。
  • 2.0 坚持早睡早起，休息的时候12点之前要睡，8:30必须起床；上班：11点半睡，7:00起床。
